# OpenClaw / Lobster Theme — Visual DNA

## Brand lineage
Clawd → Moltbot → OpenClaw. Creator: Peter Steinberger. Nov 2025 → Jan 30, 2026. Lobster mascot is sacred. Open-source Claude Code alternative, community-driven.

## Core mantras (mine these for slide copy)
- **"Your assistant. Your machine. Your rules."** — official tagline
- **"The claw is the law."** — community rallying cry
- **"New shell, same lobster soul."** — continuity through change
- **"Molting is growth."** — core principle
- **"Security by trusting a lobster."** — Peter, half-joking
- **"Fuck it, let's go with openclaw."** — Peter's 6:14am decision

## Sacred texts (filename convention from OpenClaw lore — reusable motif)
- `SOUL.md` — identity
- `memory/*.md` — long-term recall
- `AGENTS.md` — operations
- `USER.md` — creator context

## Color system (maps perfectly to Controlled Chaos `blaze` mood)
- **Bg**: `#0a0000` (ember black, almost wet tar)
- **Fg**: `#fff0e0` (cream, bisque)
- **Accent-1**: `#ff3d00` (vermillion — THE lobster red)
- **Accent-2**: `#ffab00` (gold — claw highlight, shell reflection)
- **Discord**: `#00ff88` (neon green — breaks the fire, used once for the "molt" moment)

## Typography plan
- **Headlines**: Space Grotesk (`voice-monument`) — architectural, bold
- **Code / specs**: Space Mono (`voice-brutalist`) — mechanical claws
- **Body / callouts**: DM Sans (`voice-expressive`) — soft contrast
- **Discord accent**: Google Font "Lobster" — used EXACTLY ONCE on a single glyph or word (meta-joke + breaking-own-rules principle)

## Symbolic motifs to weave through
1. **Shell shedding** — context decay / context engineering is molting
2. **Claw grip** — agents holding tools; pincer = plan + execute
3. **Heat** — blaze colors, radiant glow at edges
4. **Cocoon → butterfly** (user's original framing) — except it's molt → lobster. Transformation metaphor unifies context eng. + meta prompting
5. **Sacred texts** — SOUL.md / AGENTS.md / USER.md as a slide visual (filename stack)

## Forces tuning for presentation (custom)
The standard brutalist/immersive presets are too extreme for a presentation. Custom:
```
--force-structure: 0.75   (slide grid dominant)
--force-breath:    0.65   (legibility)
--force-warmth:    0.55   (organic, lobster energy)
--force-edge:      0.75   (sharp claws)
--force-motion:    0.4    (subtle, not chaotic)
--force-shout:     0.75   (bold headlines)
```

## The "discord" element
Controlled Chaos requires ONE rule-breaker. Two candidates:
- **Neon green lobster** that scuttles across the screen once
- **"Lobster" font glyph** dropped into a single headline as a pun (like the word "molt" rendered in Lobster)

Pick the typographic one — lower distraction, higher cleverness.
