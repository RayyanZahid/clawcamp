# ClawCamp — Vibe Coding Nights Workshop

**Facilitator:** Michalis Vasileiadis
**Session:** Agent Orchestration, Context Engineering & Meta Prompting
**Duration:** 20-min presentation + open workshop
**Theme:** OpenClaw / Lobster

## Deliverables
- HTML presentation (Controlled Chaos design system)
- GitHub repo (public)
- Vercel deployment (live URL)

## Content Pillars (order matters)
1. **Why this talk** — AI coding agents are everywhere; results are not. The gap is context.
2. **Context engineering** — the single most important skill. Include/exclude discipline, context rot, compaction, subagent isolation.
3. **Meta prompting** — letting the AI write the prompts. Self-improving loops, Anthropic's prompt tools, prompt compilation.
4. **GSD framework** — research → plan → execute → verify. How the loop optimizes both context and meta prompts.
5. **Claude Agent SDK** — building custom agents on your Claude Code subscription. The bridge from user to builder.
6. **Hands-on** — open workshop. Attendees build something live.
